Thank you for buying MasterStylizedFX.
This asset by default is in build-in render pipeline. If you are using URP, please import the MSFURP package inside this package.
If you are using URP2D, please import the URP2D package inside the package.
If you need to export to WebGL, you need to import MSFURPWebGLFix. ( due to the shader need a earlier version to support webgl.)
